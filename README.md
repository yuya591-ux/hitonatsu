# ひと夏の一日

夏の田舎で過ごす、朝から夜までの一日を味わう個人プロジェクト。
少年を歩かせて、虫を採り、住人と話し、夜は絵日記で一日を締める。
時間とともに空の色と環境音が移ろう、固定カメラの水彩風の世界。

販売・公開予定のない、完全な自己満足のための作品です。

## 技術スタック

- HTML + CSS + JavaScript（バニラ。重量級ゲームエンジンは使わない）
- 描画は Canvas API（2D）。背景・キャラ・エフェクトはすべてコードで描画する
- 音声は Web Audio API / HTML Audio（フリー素材を再生）
- ビルドは [Vite](https://vitejs.dev/)
- ホスティングは **GitHub Pages**

### なぜ GitHub Pages か

静的サイトをそのまま無料で配信でき、追加の認証設定や課金設定が不要でリモート運用と相性が良いため。
「外部APIに依存しない／自分がいなくなっても動く」という方針とも合致する。
このリポジトリでは GitHub Actions で `vite build` し、その成果物を Pages に公開している。

公開: https://yuya591-ux.github.io/hitonatsu/

## 遊び方

- 「はじめる」を押すと一日がはじまる（音もこのとき立ち上がる）。
- **歩く**: 画面をタップ／クリックした場所へ歩く。キーボードの矢印・WASD でも。
- **場面移動**: 画面の端まで歩くと、隣の場所（縁側・原っぱ・神社・田んぼ道・川辺）へ。
- **虫採り**: 虫に近づくと「つかまえる」。採った虫は左下「きろく」に残る。
- **会話**: 住人に近づくと「はなしかける」（台詞は時間帯で変わる）。
- **絵日記**: 夜に縁側で「ねる」と、その日の絵日記が出る。
- 時間で空の色・環境音・天気（朝もや/陽炎/夕焼け/夜の蛍・星・花火）が移ろう。
- スマホは横画面で。**ホーム画面に追加**するとブラウザのバーが消えて全画面（PWA）。

## ローカルでの動かし方

```bash
npm install      # 初回のみ
npm run dev      # 開発サーバを起動（表示されたURLをブラウザで開く）
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をローカルで確認
npm run verify   # ビルド＋Edge headlessで自己検証（描画/移動/虫採り/会話/日記/音）
```

URLパラメータ（調整・確認用）: `?speed=60`（早回し） `?t=0.62`（時刻固定） `?scene=jinja`（開始場所）。

## 絵を水彩画像に差し替える

背景は最初コードで描いているが、`src/assets/scenes/<場所>/fore.png` を置くと自動でその画像に差し替わる
（無ければコード描画のまま）。描き方の指針は [`ART_DIRECTION.md`](./ART_DIRECTION.md)。
音は `src/assets/audio/<音名>.mp3` を置けば差し替え可能。出典は [`CREDITS.md`](./CREDITS.md) に記録する。

## 構成（ざっくり）

- `src/engine/` … ゲームループ・時計・場面管理・記録
- `src/draw/` … 空(シグネチャ)・水彩風景・粒子・花火・仕上げ
- `src/entities/` … 主人公・虫・住人
- `src/scenes/` … 5つの場所の定義（隣接・虫・住人）
- `src/data/` … 時間帯パレット・音の対応・素材の読み込み

詳しい仕様は [`SPEC.md`](./SPEC.md)、開発の恒久ルールは [`CLAUDE.md`](./CLAUDE.md) を参照。
