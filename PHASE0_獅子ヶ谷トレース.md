# Phase 0：地形リファレンス ―「30年前の獅子ヶ谷」本格トレース（新エリア）

> 方針：**今あるエリア（town x650〜1100 ほか）は一切触らない**。実在の獅子ヶ谷の地形を入念に調べ、**別エリア（更地 x≈3000）に本格トレースで新規構築**→俯瞰スクショで合致確認→OK後に本番移行、という安全ルート。本書はその設計図（実装前の承認用）。

## 1. 実在の獅子ヶ谷の地形（調査結論・出典付き）

- **下末吉台地（標高40〜50m）**が北西に広がり、**小さな谷戸（やと）が尾根に向かって樹枝状（じゅしじょう）に何本も食い込む**＝この土地の最大の地形的個性。([鶴見区の自然環境](https://www.city.yokohama.lg.jp/tsurumi/shokai/gaiyo/sasshi/shizen.html))
- **昭和中期まで「谷戸は水田、丘陵地は森林や畑の農村地帯」**。谷底の水田は地形と不可分。([同上](https://www.city.yokohama.lg.jp/tsurumi/shokai/gaiyo/sasshi/shizen.html))
- **標高差は実在**：獅子ケ谷二丁目=6.0m（谷底）／三丁目=7.9m（横溝屋敷側）／一丁目=26.7m（台地上）。約20mの起伏。([鶴見区の地盤・ジオテック](http://www.jiban.co.jp/tips/kihon/ground/municipality/kanagawa/tsurumi/P14_tsurumi.htm))
- **横溝屋敷（殿山）＝谷戸の入口「御園（みits＝最良地）」に建つ**（獅子ヶ谷三丁目）。北へ登ると飛び地の高所へ。([みその公園・横溝屋敷](https://www.welcome.city.yokohama.jp/spot/details.php?bbid=89) / [machimori](https://machimori.main.jp/details9139.html))
- **谷の背骨＝小川（獅子ヶ谷小川アメニティ／せせらぎの小径）**が、横溝屋敷（殿山）↔ 二ツ池 ↔ 西谷広場 を結ぶ。**二ツ池〜横溝屋敷の間に“区内で唯一残る田園風景（水田）”が今もある**。([公園や自然2・鶴見区](https://www.city.yokohama.lg.jp/tsurumi/shokai/odekake/miryoku/koen_shizen/n2.html))
- **市民の森（約19ha・森の8割が北向き急斜面で夏ひんやり）**。広場が稜線沿いに5つ：旭台→灰ヶ久保→新池→西谷→下谷。池は西谷・下谷（大）・新池。([獅子ヶ谷市民の森](https://www.house.jp/areainfo/tsurumi/tsurumi-leisure/tsurumi-park/tsurumi-shishigayasiminnomorinishitanihiroba-w057-20211006/) / [西谷広場](https://iko-yo.net/facilities/161300))
- **峠道（江戸期の古道）＝西谷の谷戸と灰ヶ久保の谷戸に連なる尾根を乗っ越す**（＝車地蔵の道）。中世の鎌倉下の道が獅子ヶ谷・兜塚を通り末吉橋で鶴見川を渡る。([machimori](https://machimori.main.jp/details9139.html) / [鶴見の歴史](https://www.city.yokohama.lg.jp/tsurumi/shokai/gaiyo/sasshi/rekishi.html))
- **鶴見川は北→南へ蛇行**し南東を流れ、駒岡で矢上川と合流。([矢上川](https://ja.wikipedia.org/wiki/矢上川))

### 実在の構造（平面・北を上に・模式図）
```
     ［北西＝下末吉台地 40-50m / 鎌倉下の道］
  台地(宅地 一丁目~27m)            市民の森(丘 ~19ha)
     ┌──────────┐            ╱╲旭台広場
     │ 宅地・畑   │           ╱峠道╲灰ヶ久保     ┐
     └────┬─────┘         (車地蔵)╲新池(池)     │北向き
   西谷の谷戸│(枝谷)          ╱      ╲西谷広場(池) │急斜面
   ～～せせらぎ(小川アメニティ)～～～～～下谷広場(池)┘
        二ツ池 ●  ▒▒田園＝水田(区内唯一)▒▒
             │  谷戸の底(低 6-8m)
        横溝屋敷 ■(御園＝谷戸の入口・殿山)
```

**要約**：①北西の高い台地と、その縁に**樹枝状に食い込む谷戸**。②谷の背骨を**小川（せせらぎ）**が下り、**谷底は水田**。③横溝屋敷は**谷の入口の最良地（御園）**、二ツ池はその奥、さらに奥が**市民の森の丘（北向き斜面・5広場・池）**。④**尾根の峠道＝車地蔵**。⑤南東の低地を**鶴見川**。

## 2. 新エリアへの座標対応（更地 x≈3000・既存に非接触）

エリアは x オフセットで分離（field x≈0／town x=1000／shrine x=2000）。**新エリア『獅子ヶ谷（本格トレース）』を NX=3000 中心**に置く。歩行範囲の目安 **x[2850, 3150]／z[-140, 150]**（約300×290m）。**town(x650〜1100)とは霧と距離で完全分離**。

向きの約束（このエリア）：**+z＝北＝谷の口＝低い（鶴見川側）／−z＝南＝谷の奥＝森・台地＝高い**（townと同じ感覚で扱える）。

### 樹枝状（じゅしじょう）の谷戸を作る＝この土地の核心
- 主谷戸（背骨）：横溝屋敷→二ツ池→西谷広場→森、を**せせらぎが下る一本の谷底**。
- **枝谷を2〜3本**、主谷から台地・森へ食い込ませる（短い指状のくぼみ＋その底に小さな田）。これで「尾根に向かって谷が何本も入り込む」実在の特徴を地形で再現。

## 3. 地形の作り方（heightAt の“本格”モデル＝ad-hocなsmoothstep積みではない）

新エリア専用の `heightAtYato(x,z)` を**谷の背骨モデル**で構築（保守可能・忠実）：
1. **谷底センターライン**（横溝→二ツ池→西谷の折れ線）と、線に沿う**谷底標高**（口=低→奥=高）。
2. 点(x,z)から**最近点までの横距離 d** を出し、`floorH(spine) + rise(d)` で谷壁を立ち上げる（d小=谷底平ら、d大=尾根/台地へ上昇）。
3. **市民の森＝ガウシアンの丘**（−z側・北向きを意識）＋**台地＝西側(−x)の高い平面**。
4. **枝谷**＝副次センターラインを2〜3本足し、台地/森に切れ込みを入れる（樹枝状）。
5. **峠道の鞍部**＝尾根上の一点を少し下げて“越え”を作る。
※ 既存 `heightAt` は触らず、`if (x>2600) return heightAtYato(x,z)` の新分岐を足すだけ＝**完全に追加・非破壊**。
