## 0. 大原則（毎タスクで必ず守る）

1. **3視点で必ず確認する**：
   - **上から（俯瞰／ドローン）**：配置・色面・繰り返しパターン・密度。
   - **中位の高さ（散歩・座る・寝ころぶ）**：シルエット・近景の質感・空の大きさ。
   - **主人公目線（間近〜中距離）**：接地・手触り・逆光・被写界。
   どれか一つで良く見えても、他で破綻していたら未完。
2. **エモ最優先**：様式（低ポリ・トゥーン）はエモさのためなら崩してOK（ユーザー許可済）。ただし雑に崩さない・冷たいフォトリアル全振りは禁止。温かい絵本／水彩の空気を守る。
2.5. **⚠️演出は「最高峰」か、やらない（ユーザー強い指摘2026-06-25）**：霧・光・色補正・DOF等の“雰囲気演出”は、中途半端だと逆効果（例＝遠景が灰色のモヤに濁って見えた）。**自然に見えるまで徹底的に詰める**。霧なら「空の地平色に合わせ・明るく・距離はゆるやかに溶ける」＝灰色の濁りでなく空に溶ける空気遠近。この基準は**全要素に共通**で適用する。
3. **小さく重ね塗り**：1イテレーション＝1つの小さな改善＋検証＋コミット。「完成」で止めない。
4. **検証してから言い切る**：`npm run build`→該当の `scripts/_*.mjs` 探針で複数アングルのスクショ→批判的に自分で見る→`npm run verify`（コンソールエラー0）。都合のいい絵で成功を断言しない。
5. **回転規約に注意**：建物の壁/屋根/看板/当たり判定の回転の向きを混ぜない（[[geometry-rotation-conventions]]）。
6. **描画負荷を見張る**：インスタンス/マージ/LODを使い、ドロー数を増やしすぎない。品質優先だが破綻しない範囲で。
7. **著作権**：実在ロゴ・固有名の台詞・原作の絵/音は模倣しない。普遍的な「あの夏の空気」を作る。

## 柱A：光・空気・マジックアワー（エモさの最大レバー）

現状：朝昼夕夜の色補正・霧・リム・ゴッドレイ・ブルーム・インク線・記憶トーンは高水準。夕→夜の移ろいが白眉。伸びしろは「夕日の濃密化」と「被写界・露出」。

- [x] A1. **被写界深度(DOF)を導入**（2026-06-25・ユーザー承認「弱めに」）：既存の深度テクスチャ(normalRT.depthTexture)を再利用する自前DOFパス(ShaderPass)を gradePassの後・inkPassの前に挿入＝色だけ柔らかく**輪郭線はくっきり保つ**＝温かいトゥーンを壊さない。フォーカス＝画面中央やや下(主人公)・ピント面から離れた遠景/最近景だけ8タップで弱くぼかす(maxCoc2.4px・空でも全面ボケしないようfz頭打ち110m)。深度RTはink/dofのどちらかONで描画・リサイズで貼り直し。`_dof`フックで調整可。検証`_dofcheck.mjs`(off/on比較＝遠景の町が柔らかく/主人公はくっきり)。**強すぎたら_dofでstrength/maxCocを下げる**。
- [x] A2. **マジックアワーの濃密化**（2026-06-25）：gradePassのgolden項に**燃える地平の橙**(下ほど強い)を追加＋金色の押しを少し強く、上空は紫みを残して奥行き。goldenの窓を少し早く長く(0.56→0.94)＝マジックアワーを長く味わう。duskパレットの輪郭の橙ふち(rim 0xff6f24/ri1.45)と地平の金橙(bot 0xeaa274)を強化、上空は richer violet(0x645592)。検証`_dusk.mjs`(0.64/0.74/0.80・夕日の方角を向いて撮影)。昼夜は不変(golden項はduskのみ)。
- [ ] A3. **逆光のにじみ／レンズフレア**：太陽が画面に入った時の柔らかいハレーション（ブルームと別の、絵本的な光芒）。やり過ぎ注意。
- [~] A4. **空気遠近の強調**（2026-06-25・ユーザー指摘「遠景にモヤ」対応）：昼/朝の霧色が灰色で空より暗く＝遠景が灰色の濁りになっていた→**霧色を明るい空色に寄せ(noon 0xe3eae8→0xe9eff3・morn→0xe4edf4)＋霞の始まりを75→105・到達を460→560へ奥に**＝中景はくっきり残り遠景だけ空にやわらかく溶ける自然な空気遠近に。検証`_fog.mjs`(住宅街/原っぱから遠景の溶け方)。**残り：遠景の彩度をもう少し落として薄青へ・夕/夜の霞も同基準で点検**。
- [ ] A5. **木漏れ日を顔・地面に落とす**：今は静的な加算円。葉影のテクスチャを動かしてキャラ/地面に落ちる斑へ。
- [ ] A6. **夕立あがりの濡れ**：雨後に道・屋根・葉が少し濡れて照り返す（一時的なスペキュラ／色）。虹とセットで。
- [ ] A7. **虹の位置を太陽と整合**：太陽の反対側・対日点42°に架ける計算へ。二重虹のうっすら外側も。
- [~] A8. **夜の濃さと安らぎ**：月明かりの青、窓あかりの暖色の対比をもう一段。星をもう少し瞬かせる／天の川のうっすら。 ／ **星空(2026-06-25)**＝四角い点→やわらかい丸（放射グラデのテクスチャ）＋星数を増やして満天に(480→740)＋天の川の帯を少し締める。検証`_stars.mjs`(深夜0.96で見上げ/中天＝丸い星が密に・天の川の帯)。 ／ **星の瞬き(2026-06-26)**＝PointsMaterialを自前ShaderMaterialへ（蛍の手法を流用）。星ごとに位相/速さ/基準輝度/大きさ/ほのかな色み(白多め・一部 青白/暖白)をばらし、`sin(uTime*aSpeed+aPhase)`で輝度と大きさがゆっくり揺れる＝満天がちらちら瞬く。夜のフェードはuOpacity=nfで従来通り・layer1でインク線除外。`_twinkle.mjs`で空の明るい画素数が時間で変動(1508→1924)＝またたきを定量確認・全体の明るさは満天を保持。**残り：月明かりと窓の暖色の対比強化**。
