# 場面の絵の差し替え（水彩画像）ガイド

ここに画像ファイルを置くだけで、その場面が「コードで描いた版」から「絵を貼った版」に
**自動で**切り替わります（ビルド時に拾われます）。置かなければコード描画のまま。404 も出ません。
Gemini などで描いた水彩画を、後からここに置いて格上げするための仕組みです。

## いちばん簡単：場面まるごと一枚の絵に差し替える（おすすめ）

```
src/assets/scenes/<場面ID>/scene.png
```

- `scene` という名前の画像を置くと、その場面の遠景・地面・前景の**コード描画を丸ごと置き換え**ます。
- 主人公・住人・虫・猫・UI・時間帯の色は、絵の**上に自動で乗ります**（差し替え後もゲームはそのまま動く）。
- 例: `src/assets/scenes/harappa/scene.png` を置く → 原っぱがその一枚絵になる。
- テンプレート例（水彩風のサンプル）: リポジトリの `docs/art-swap-example.svg` を参照。

## 部分的に差し替えたいとき（層ごと）

```
src/assets/scenes/<場面ID>/<レイヤー名>.png
```

- レイヤー名: `far`（遠景の山）/ `ground`（地面）/ `fore`（手前の主役）/ `sky`（空も絵にしたいとき）
- 必要な層だけ置けば、その層だけ画像になります（残りはコード描画）。

## 形式・サイズ

- 形式: `.png` `.jpg` `.webp` `.svg`
- 画面いっぱいに引き伸ばして表示します。**横長 16:9** で用意してください（推奨 1920×1080 以上）。
- 主人公が歩く帯（画面の下〜中ほど）に大事なもの（人物・障害物）を描き込みすぎないこと。
  主人公や虫はその上を歩くので、地面・道として成立する絵にします。

## 時間帯（朝昼夕夜）について ― ここが大事

- **絵は「昼の素直な色」で1枚描けばOK**です。朝焼け・夕焼け・夜の藍は、
  仕上げ（WebGLシェーダー＋ポスト処理）が**自動で上から色を乗せて**一日を表現します。
- もし時間帯ごとに別の絵を使いたくなったら、将来 `scene-morning` / `scene-evening` のような
  命名で差し替えられるよう拡張できます（今は単一の `scene` を基準色＝昼で運用）。

## 仕組み（参考）

- 拾う処理: `src/data/assets.js`（Vite の `import.meta.glob` で存在するファイルだけ収集）
- 切り替え: `src/draw/layer.js`（画像が読めたら `drawImage`、ダメならコード描画にフォールバック）
- 場面の層構成: `src/scenes/scenes.js`（`scene` 画像があれば far/ground/fore を置き換え）
