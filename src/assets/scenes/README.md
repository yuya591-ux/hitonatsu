# 場面の差し替え画像の置き場

ここに画像を置くと、その場面のそのレイヤーが「コード描画版」から「画像を貼る版」に自動で切り替わります（ビルド時に拾われます）。置かなければコード描画のままです。

## 置き方

```
src/assets/scenes/<場面ID>/<レイヤー名>.png
```

- 場面ID: `engawa`（縁側）/ `harappa`（原っぱ）/ `jinja`（神社）/ `tanbomichi`（田んぼ道）/ `kawabe`（川辺）
- レイヤー名: `far`（遠景の山）/ `ground`（地面）/ `fore`（手前の主役）
- 形式: `.png` `.jpg` `.webp`

例: `src/assets/scenes/engawa/fore.png` を置くと、縁側の前景（縁側そのもの）が画像になります。

## 補足

- 空（空グラデ＋雲＋光＝シグネチャ表現）は時間で動くため、画像差し替えの対象外です。
- 画像は画面いっぱいに引き伸ばして表示します。横長（16:9 など）で用意してください。
- Gemini で作った水彩画などを、後からここに置いて格上げできます。
